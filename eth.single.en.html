    Create Wallet

    import requests

requests.post(
    url="https://testnet.sectoken.io/wallet/eth_single/create/", 
    headers={"HMAC": "your token"}, 
    json={
        "pubkey": "03976717038a5c6967535cc469acb6d7a43dee00568b9c4d4f2bbccb287503787c",
        "wallet_name": "your wallet name",
    }
)

    curl "https://testnet.sectoken.io/wallet/eth_single/create/" \
    -H "HMAC:your token" \
    -H "Content-Type:application/json" \
    -X POST \
    -d '{"pubkey": "03976717038a5c6967535cc469acb6d7a43dee00568b9c4d4f2bbccb287503787c", "wallet_name": "your wallet name"}'

    Response Samples

    {
    "code": 200,
    "data": {
        "wallet_id": "cf761ad1b6c5f30a1557d942cb557b07",
        "create_time": "2019-10-10T10:13:03.195",
        "activate_code": "712968"
    }
}

    HTTP Request

    POST https://testnet.sectoken.io/wallet/eth_single/create/

    Request Parameters

    Parameter Name Type Required Description
    pubkey string Yes Compressed public key. Length of 66.
    privkey string No Private key. Users should encrypt and decrypt when using. (Must be base64 format.)
    wallet_name string Yes Wallet Name

    Response Parameters

    Parameter Name Type Description
    code integer Status code. Successful return 200. Others are error codes.
    data object Response data.

    Data Parameters

    Parameter Name Type Description
    wallet_id string Wallet id
    create_time string Creation time of wallet
    activate_code string Activate code

    HTTP Request

    GET https://testnet.sectoken.io/wallet/eth_single/detail/

    Request Parameters

    Parameter Name Type Required Description
    wallet_id string Yes Wallet id
    coin_type string No Coin type. ETH or other ERC20 token supported by the platform. Default is "ETH".

    Response Parameters

    Parameter Name Type Description
    code integer Status code. Successful return 200. Others are error codes.
    data object Response data.

    Data Parameters

    Parameter Name Type Description
    wallet_id string Wallet id
    wallet_name string Wallet Name
    address string Eth address of wallet
    balance string Balance. (Minimum unit)
    send_max string Maximum amount per transfer. "0" is unlimited.
    send_day_limit integer Maximum number of transfers per day. 0 is unlimited.
    send_day_max string Maximum number of transfers per day. "0" is unlimited.
    freeze_overtime integer Wallet freeze deadline. 0 is not frozen.

    HTTP Request

    POST https://testnet.sectoken.io/wallet/eth_single/dotx/

    Request Parameters

    Parameter Name Type Required Description
    wallet_id string Yes Wallet id
    to_address string Yes Destination address.
    value string Yes Transfer amount. (Minimum unit).
    coin_type string No Coin type. ETH or other ERC20 token supported by the platform. Default is "ETH".
    feerate int No Transfer speed. 1 is fast, 0 is normal. Default is 0.
    memo string No This only works if the coin_type is ETH。
    total int No Transfer all balance. set value 1 to enbale. The value parameter is not in effect at this point.
    Filling in memo will cause extra fee

    Response Parameters

    Parameter Name Type Description
    code integer Status code. Successful return 200. Others are error codes.
    data object Response data.

    Data Parameters

    Parameter Name Type Description
    unsigned_tx string Unsigned transaction
    privkey string The privkey field that was submitted when the wallet was created. Uncommitted returns an empty string.

    Transaction Callback

    When a transaction occurs in a wallet，Sectoken will send an Http POST request to the specified url，The content-type of the request header is application/json.。

    This url is specified by the user, who must deploy the service to receive the HTTP POST request.

    code integer Status code. Successful return 200. Others are error codes.
    data array Response data

    Data Parameters

    Parameter Name Type Description
    txid string Transaction id
    src_address string Transaction source address
    det_address string Transaction destination address
    amount string Transfer amount
    timestamp string Transfer time
    state int Transfer status, 0 pay 1 receive
    height int Block height
    decimal int Coin balance decimal places
