    创建eth多签钱包

    import requests

requests.post(
    utl="https://example.com/wallet/eth/getkey/", 
    headers={"HMAC": "your token"}, 
    json={
        "pubkeys": ["compressed pubkey1",...],
        "wallet_name": "your wallet name",
        "multi_type": "2-3",
        "order_type": 0,
    }
)


    
curl "https://example.com/wallet/eth/getkey/"
  -H "HMAC:your token"
  -H "Content-Type:application/json"
  -X POST
  -d '{"pubkeys": ["compressed pubkey1",...], \
      "wallet_name": "your wallet name", \
      "multi_type": "2-3", \
      "order_type": 0}'

    HTTP Request

    GET https://example.com/wallet/eth/getkey/

    请求参数

    参数名 参数类型 必填 说明
    pubkeys array 是 公钥数组
    privkeys array 否 私钥数组
    wallet_name string 是 钱包名
    password string 否 钱包密码
    multi_type string 是 多签类型，格式为 {需要的签名数}-{总签名数}
    order_type int 是 平台是参与签名，0：平台参与签名，1：平台不参与签名

    返回参数

    参数名 参数类型 说明
    code int 返回码，200成功，其他失败，请查看错误码
    data object 返回内容

    data 参数

    参数名 参数类型 说明
    wallet_name string 钱包名
    wallet_id string 钱包的id
    create_time string 钱包创建时间
    user_key_codes array 私钥数组
    pubkey_codes array 公钥数组
    activate_code string 钱包激活码
    server_pub_key string 钱包的xpub

    错误码

    当发生错误时返回的错误码

    错误码 错误说明
    4203 费用地址余额不足
    4401 账户认证失败
    4403 钱包个数达到上限
    4602 当前账户没有费用地址
    500 未知错误
