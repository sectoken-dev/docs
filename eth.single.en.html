    Create Wallet

    import requests

requests.post(
    url="https://testnet.sectoken.io/wallet/eth_single/create/", 
    headers={"HMAC": "your token"}, 
    json={
        "pubkey": "compressed pubkey",
        "wallet_name": "your wallet name",
    }
)

    curl "https://testnet.sectoken.io/wallet/eth_single/create/" \
    -H "HMAC:your token" \
    -H "Content-Type":"application/json" \
    -X POST \
    -d '{"pubkeys": "compressed pubkey1", "wallet_name": "your wallet name"}'

    Response Samples

    {
    "code": 200,
    "data": {
        "wallet_id": "cf761ad1b6c5f30a1557d942cb557b07",
        "create_time": "2019-10-10T10:13:03.195",
        "active_code": "712968"
    }
}

    HTTP Request

    POST https://testnet.sectoken.io/wallet/eth_single/create/

    Request Parameters

    Parameter Name Type Required Description
    pubkey string Yes Compressed public key.
    privkey string No Private key. Users should encrypt and decrypt when using.
    wallet_name string Yes Wallet Name
    password string No Recovery password. No wallet password.

    Response Parameters

    Parameter Name Type Description
    code integer Status code. Successful return 200. Others are error codes.
    data object Response data.

    Data Parameters

    Parameter Name Type Description
    wallet_id string Wallet id
    create_time string Creation time of wallet
    activate_code string Activate code

    HTTP Request

    GET https://testnet.sectoken.io/wallet/eth_single/activate/

    Request Parameters

    Parameter Name Type Required Description
    wallet_id string Yes Wallet id
    coin_type string No Coin type. ETH or other ERC20 token supported by the platform. Default is "ETH".

    Response Parameters

    Parameter Name Type Description
    code integer Status code. Successful return 200. Others are error codes.
    data object Response data.

    Data Parameters

    Parameter Name Type Description
    wallet_id string Wallet id
    wallet_name string Wallet Name
    address string Eth address of wallet
    balance string Balance. (Minimum unit)
    send_max string Maximum amount per transfer. "0" is unlimited.
    send_day_limit integer Maximum number of transfers per day. 0 is unlimited.
    send_day_max string Maximum number of transfers per day. "0" is unlimited.
    freeze_overtime integer Wallet freeze deadline. 0 is not frozen.

    Transaction Callback

    When a transaction occurs in a wallet，Sectoken will send an Http POST request to the specified url，The content-type of the request header is application/json.。

    This url is specified by the user, who must deploy the service to receive the HTTP POST request.

    request body

    {
    "wallet_id": "6004e156344ec27cad24a8df5bbaa5ab",
    "txid": "0x2b4675aff7efad0c7dcee09a943f9cc501402f18b67393a47cdcfa887f5ab2da",
    "height": 4340000,
    "dst_address": "0x759ac48a2754be136d495da1a826200ab1c911f2",
    "src_address": "0x8cb9e65d6a771424dfabbda4221bdb446d9183fe",
    "state": 1,
    "timestamp": 1872340000,
    "amount": "10000000000",
    "wallet_balance": "100000000000",
    "coin_type": "ETH",
    "decimal": 18
}

    Parameters descriptions

    Parameter Name Type Description
    wallet_id string wallet_id
    txid string Transaction id
    height int Block height
    dst_address string Destination address
    src_address string Source address
    state int Transfer status, 0 pay 1 receive
    timestamp int Block confirmation time
    amount string Transfer amount
    wallet_balance string Coin type balance in wallet
    coin_type string coin type
    decimal int Coin balance decimal places

    Error Codes

    The error code returned when an error occurs

    Error code Description
    400 Invalid params
    500 Unknown error
    4203 Insufficient Balance
    4204 Wallet frozen
    4207 Wallet already exist
    4302 The amount of a single transaction exceeds the limit
    4303 The trading amount exceeded the upper limit on the day
    4304 The number of trades exceeded the limit
    4305 The destination address is not on the white list
    4401 Authentication failed
    4403 The number of wallets reached the limit
    4507 Token doesn't exist
    4509 Signature error
    4601 The number of derived addresses reaches an upper limit
    4603 Can't transfer it to yourself
