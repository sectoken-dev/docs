    创建eth单签钱包

    import requests

requests.post(
    url="https://testnet.sectoken.io/api/v1/wallet/eth_single/create/", 
    headers={"HMAC": "your token"}, 
    json={
        "pubkey": "03976717038a5c6967535cc469acb6d7a43dee00568b9c4d4f2bbccb287503787c",
        "wallet_name": "EthSingleTestWallet",
    }
)

    curl "https://testnet.sectoken.io/api/v1/wallet/eth_single/create/" \
    -H "HMAC:your token" \
    -H "Content-Type:application/json" \
    -X POST \
    -d '{
        "pubkey": "03976717038a5c6967535cc469acb6d7a43dee00568b9c4d4f2bbccb287503787c",
        "wallet_name": "EthSingleTestWallet"
    }'

    var request = require('request');

request({
    url: "https://testnet.sectoken.io/api/v1/wallet/eth_single/create/",
    method: "post",
    headers: {
        'Content-Type': 'application/json',
        'HMAC': 'your token',
    },
    body: JSON.stringify({
        'pubkey': '02c21fa2c5e7f10ee87b22dd33dc1e6eabe3416a9cb117e4f344d590eda7fd3088',
        'wallet_name': 'EthSingleTestWallet'
    })
});

    返回示例

    {
    "code": 200,
    "data": {
        "wallet_id": "cf761ad1b6c5f30a1557d942cb557b07",

}

    HTTP Request

    POST https://testnet.sectoken.io/api/v1/wallet/eth_single/create/

    请求参数

    参数名 参数类型 必填 说明
    pubkey string 是 压缩公钥，长度为66
    privkey string 否 私钥，用户应自行加密传输，使用时解密。（必须使用base64编码）
    wallet_name string 是 钱包名

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data object 返回内容

    data 参数

    参数名 参数类型 说明
    wallet_id string 钱包的id
    activate_code string 钱包激活码

    HTTP Request

    POST https://testnet.sectoken.io/api/v1/wallet/eth_single/dotx/

    请求参数

    参数名 参数类型 必填 说明
    wallet_id string 是 钱包id
    to_address string 是 收款地址
    value string 是 转账金额，最小单位
    coin_type string 否 币种，默认为eth，可以指定为平台支持的erc20 token，如erc20_ht，erc20_koz
    gas_pr integer 否 转账速度，1快速，0普通，默认为0
    total boolean 否 全部转出，true生效，此时value值不生效。
    gas_price integer 否 gas price, 为0时系统自动分配合适的price, 默认为0

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data object 返回内容

    data 参数

    参数名 参数类型 说明
    unsigned_tx string 未签名的交易
    privkey sting 创建钱包时提交的privkey字段，未提交则返回空字符串
    order_id integer 构建交易单时返回的订单id

    HTTP Request

    POST https://testnet.sectoken.io/api/v1/wallet/eth_single/batch_dotx/

    请求参数

    参数名 参数类型 必填 说明
    wallet_id string 是 钱包id
    outputs array 是 转账目标地址及金额信息
    coin_type string 否 币种，默认为eth，可以指定为平台支持的erc20 token，如erc20_ht，erc20_koz
    feerate integer 否 转账速度，1快速，0普通，默认为0

    Outputs 请求参数

    参数名 参数类型 必填 说明
    address string 是 目标地址
    value string 是 转账金额

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data object 返回内容

    data 参数

    参数名 参数类型 说明
    approval_tx string 授权交易单的信息, 当转 ETH时或ERC20授权足够时,此项为空字符串
    unsigned_tx string 未签名的交易
    privkey string 创建钱包时提交的privkey字段，未提交则返回空字符串

    交易Callback通知

    当用户钱包发生交易时，会向指定的url发送HTTP POST请求，请求头content-type为application/json。

    此url由用户指定，用户须部署服务接收HTTP POST请求。同时需先添加钱包Webhook。

    请求头中会有Checksum参数，用于验证请求信息是否完整， 通过 sha256（secret_key+request.body）来进行哈希.

    secret_key 为创建webhook时提交的secret_key, 如果创建时没有上传,默认为空字符串.

    请求headers示例 json { "Content_type": "application/json", "Checksum": "sha256=6ca13d52ca70c883e0f0bb101e425a89e8624de51db2d2392593af6a84118090" }
