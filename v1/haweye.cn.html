    登录接口

    curl "https://testnet.sectoken.io/api/v1/haweye/accounts/login/" \
    -H "Content-Type:application/json" \
    -X POST \
    -d '{
    "username": "test",
    "password":"test123456"
}'

    返回示例

    {
  "code": 200
}

    HTTP Request

    POST https://testnet.sectoken.io/api/v1/haweye/accounts/login/

    请求参数

    参数名 参数类型 必填 说明
    username string 是 用户名
    password string 是 密码

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码

    获取实时监控

    import asyncio
import websockets


async def hello():
    url = "ws://testnet.sectoken.io/words/push"
    async with websockets.connect(url) as websocket:
        while True:
            greeting = await websocket.recv()
            print("收到: ", greeting)


if __name__ == '__main__':
    asyncio.get_event_loop().run_until_complete(hello())
