    HTTP Request

    POST https://testnet.sectoken.io/api/v1/wallet/trx/create/

    请求参数

    参数名 参数类型 必填 说明
    pubkeys array 是 公钥列表
    privkeys array 否 私钥列表，私钥必须编码成base64格式
    wallet_name string 是 钱包名

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data object 返回内容

    data 参数

    参数名 参数类型 说明
    wallet_id string 钱包的id
    activate_code string 钱包激活码
    server_pubkey string 钱包的hd xpub
    address string 钱包的地址

    创建多用户钱包

    创建一个任意多签类型的多用户钱包, 您可以通过邀请码来邀请他人加入您的钱包, 并参与交易签名

    import requests

requests.post(
        url="https://testnet.sectoken.io/api/v1/wallet/trx/share_create/",
        headers={"Authorization": "Bearer Atk38e9b38c64430d56ee34a05f485279ae"},
        json={
            "pubkey": "027c9612b91ed830de2259819c85e160084fb1fb0ca5ad2d02537c085f3b93246c",
            "privkey": "NGRjMjdkYWZkYjY3YWY2Y2M4MDNiNWMwYWQyMjIxNzlmMjlkZmNkNWM5NmVhNzY5NTRlNTUxZDY1M2Q3NGM3Zg==",
            "wallet_name": "TestWallet",
            "threshold": 2,
            "limit": 2,
            "control": 1,
        }
)

    curl "https://testnet.sectoken.io/api/v1/wallet/trx/share_create/" \
    -H "Authorization: Bearer $BEARER_TOKEN" \
    -H "Content-Type:application/json" \
    -X POST \
    -d '{
        "pubkey":  "027c9612b91ed830de2259819c85e160084fb1fb0ca5ad2d02537c085f3b93246c",  
        "privkey":"NGRjMjdkYWZkYjY3YWY2Y2M4MDNiNWMwYWQyMjIxNzlmMjlkZmNkNWM5NmVhNzY5NTRlNTUxZDY1M2Q3NGM3Zg==", 
        "wallet_name": "TestWallet",
        "threshold": 2,
        "limit": 2,
        "control": 1,
    }'

    let request = require("request");

request({
    url: "https://testnet.sectoken.io/api/v1/wallet/trx/share_create/",
    method: "post",
    headers: {
        "Content-Type": "application/json",
        "Authorization": "Bearer Atk38e9b38c64430d56ee34a05f485279ae",
    },
    body: JSON.stringify({
        "pubkey": "027c9612b91ed830de2259819c85e160084fb1fb0ca5ad2d02537c085f3b93246c",
        "privkey": "NGRjMjdkYWZkYjY3YWY2Y2M4MDNiNWMwYWQyMjIxNzlmMjlkZmNkNWM5NmVhNzY5NTRlNTUxZDY1M2Q3NGM3Zg==",
        "wallet_name": "TestWallet",
        "threshold": 2,
        "limit": 2,
        "control": 1,
    })
});

    返回示例

    {
  "code": 200,
  "data": {
    "invitation_code": "cf761ad1b6c5f30a1557d942cb557b07"
  }
}

    HTTP Request

    POST https://testnet.sectoken.io/api/v1/wallet/trx/share_create/

    请求参数

    参数名 参数类型 必填 说明
    pubkey string 是 公钥
    privkey string 否 加密后并进行base64编码后的私钥
    wallet_name string 是 钱包名
    threshold integer 是 钱包最少签名个数,最大为7
    limit integer 是 钱包公钥总个数,最大为7
    control integer 是 0:平台参与签名; 1:平台不参与签名
    multi integer 否 是否允许单用户持有多个公钥，允许为1，默认值为0

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data object 返回内容

    data 参数

    参数名 参数类型 说明
    invitation_code string 钱包邀请码

    激活钱包

    通过快速创建钱包或者加入共享钱包所返回的 wallet_id 和 activate_code 来激活钱包,激活后钱包可以进行正常操作 必须先向钱包地址中充值才能激活钱包

    import requests

requests.post(
        url="https://testnet.sectoken.io/api/v1/wallet/trx/activate/",
        headers={"Authorization": "Bearer Atk38e9b38c64430d56ee34a05f485279ae"},
        json={
            "wallet_id": "cf761ad1b6c5f30a1557d942cb557b07",
            "activate_code": "712968",
        }
)

    curl "https://testnet.sectoken.io/api/v1/wallet/trx/activate/" \
    -H "Authorization: Bearer $BEARER_TOKEN" \
    -H "Content-Type:application/json" \
    -X POST \
    -d '{
            "wallet_id": "cf761ad1b6c5f30a1557d942cb557b07", 
            "activate_code": "712968"
    }'

    let request = require("request");

request({
        url: "https://testnet.sectoken.io/api/v1/wallet/trx/activate/",
        method: "post",
        headers: {
            "Content-Type": "application/json",
            "Authorization": "Bearer Atk38e9b38c64430d56ee34a05f485279ae",
        },
        body: JSON.stringify({
            "wallet_id": "cf761ad1b6c5f30a1557d942cb557b07",
            "activate_code": "712968"
        })
    }
);

    HTTP Request

    GET https://testnet.sectoken.io/api/v1/wallet/trx/detail/

    请求参数

    参数名 参数类型 必填 说明
    wallet_id string 是 钱包id

    参数名 参数类型 说明
    wallet_id string 钱包id
    wallet_name string 钱包名
    address string 钱包地址
    balance string 钱包余额，单位为最小单位
    net string 剩余的免费带宽
    threshold integer 钱包最少签名个数,最大为7
    limit integer 钱包公钥总个数,最大为7
    send_max string 单次转账上限，"0"为无上限
    send_day_limit integer 单日转账次数上限，0为无上限
    send_day_max string 单日转账上限，"0"为无上限
    freeze_overtime integer 钱包冻结截止时间，0为未冻结
    approval_txs array 待签名的订单id, 只有多用户钱包有此项

    交易Webhook通知

    当用户钱包发生交易时，会向指定的url发送HTTP POST请求，请求头content-type为application/json。

    此url由用户指定，用户须部署服务接收HTTP POST请求。同时需先添加钱包Webhook。

    请求头中会有Checksum参数，用于验证请求信息是否完整， 通过 sha256（secret_key+request.body）来进行哈希.

    secret_key 为创建webhook时提交的secret_key, 如果创建时没有上传,默认为空字符串.

    请求headers示例

    {
  "Content_type": "application/json",
  "Checksum": "sha256=6ca13d52ca70c883e0f0bb101e425a89e8624de51db2d2392593af6a84118090"
}

    请求body示例

    {
  "wallet_id": "cf761ad1b6c5f30a1557d942cb557b07",
  "txid": "2b4675aff7efad0c7dcee09a943f9cc501402f18b67393a47cdcfa887f5ab2da",
  "height": 4340000,
  "src_address": "TXorxEGAgkMbbKy4HRjtuCZcDsfXmdz55n",
  "outputs": [
    {
      "dst_address": "TVh9fBP1eFFdUZR325XdsdnQqcWYbp5F5e",
      "value": "1000000000"
    }
  ],
  "type": "deposit",
  "timestamp": 1872340000,
  "coin": "trx",
  "decimal": 6,
  "webhook_id": "fa5dc2025e79c37b8656ef4b89619d9d"
}

    参数说明

    参数名 参数类型 说明
    wallet_id string 钱包id
    txid string 交易单id
    height integer 交易所在的区块高度
    src_address string 交易转出地址
    outputs array 收款地址及金额
    type string 交易类型,收款或付款
    timestamp integer 交易所在区块生成时间
