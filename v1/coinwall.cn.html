    获取USDT充值地址

    curl "https://coinwall.sectoken.io/recharge_address/?coin=eth" \
    -H "Authorization: Bearer $BEARER_TOKEN" \

    返回示例

    {
  "code": 200,
  "data": {
    "address": "0x66a977F4ecB64972c79Fe2c220326D0aE7e600aF"
  }
}

    HTTP Request

    GET https://coinwall.sectoken.io/recharge_address/

    请求参数

    参数名 参数类型 必填 说明
    coin string 否 币种信息, 目前仅支持 eth

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data object 返回内容

    data 参数

    参数名 参数类型 说明
    address string USDT 充值地址

    获取商品列表

    curl "https://coinwall.sectoken.io/goods_list/" \
    -H "Authorization: Bearer $BEARER_TOKEN"

    返回示例

    {
  "code": 200
}

    HTTP Request

    GET https://coinwall.sectoken.io/goods_list/

    请求参数

    无需请求参数

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data object 返回内容

    data参数

    参数名 参数类型 说明
    goods_id integer 商品 id
    title string 商品名称
    price string 商品价格
    broadcast_limit integer 广播次数
    filter_limit integer 规则条数
    level integer VIP 等级
    info string 商品描述

    HTTP Request

    GET https://coinwall.sectoken.io/vip_info/

    请求参数

    无需请求参数

    返回参数

    参数名 参数类型 说明

    购买 VIP 商品

    curl "https://coinwall.sectoken.io/buy_vip/" \
    -H "Authorization: Bearer $BEARER_TOKEN" \
    -H "Content-Type:application/json" \
    -X POST \
    -d '{"goods_id": 1, "num":3}'

    返回示例

    {
  "squence_id": "2010280000159627",
  "vip": {
    "broadcast_limit": 3000,
    "filter_limit": 15,
    "end_time": "2021-01-28T00:00:00Z",
    "vip_level": 1,
    "method": "buy",
    "price": "15.0 USDT",
    "balance": "5.0 USDT"
  }
}

    HTTP Request

    POST https://coinwall.sectoken.io/buy_vip/

    请求参数

    参数名 参数类型 必填 说明
    goods_id string 是 商品id
    num integer 是 购买数量

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data object 返回内容

    data参数

    参数名 参数类型 说明
    broadcast_limit string 剩余广播次数
    filter_limit string 监控规则条目
    end_time string VIP 结束时间
    vip_level integer 当前 VIP 等级
    method string 购买类型 (购买/续费)
    price string 消费金额 USDT为单位
    balance string 剩余余额 USDT为单位

    获取购买记录

    curl "https://coinwall.sectoken.io/payment_record/" \
    -H "Authorization: Bearer $BEARER_TOKEN" \  
    -x POST
    -d '{"vip_level":1, "page":1}'

    返回示例

    {
  "code": 200,
  "data": [
    {
      "sequence_id": "db4f57bc1c9f339bd142bd067ed02d5b",
      "vip_level": 1,
      "value": "5.0 USDT",
      "buy_num": 3,
      "total_amount": "15.0 USDT",
      "c_time": "2020-10-27T00:00:00Z"
    }
  ]
}

    HTTP Request

    GET https://coinwall.sectoken.io/payment_record/

    请求参数

    参数名 参数类型 必填 说明
    vip_level string 否 监控地址
    page integer 否 页码, 默认为 1

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data array 返回内容

    data 参数

    参数名 参数类型 说明
    sequence_id string 订单 id
    vip_level integer vip 等级
    value string 单价
    buy_num integer 购买数量
    total_amount string 支付总金额
    c_time string 创建时间

    创建规则集

    curl "https://coinwall.sectoken.io/api/v1/wall/filter" \
    -x POST \
    -H "Authorization: Bearer $BEARER_TOKEN" \  
    -d "{
      'name':'btc_test',
      'coin':'btc',
      'address':'2N4mEcHPeKNcz5jmX1vzrsVhqffb8BCzH9L',
      'conditions':[
        {
          'name': 'black_list', 
          'values': ['2N7LGJUpCqufDE4NLjecHSimHCDrFwQsgrk', '2N2587eHcUCXpfjJ2B5bHQYNuijRtCXKpmi']
        }
      ]
    }"

    返回示例

    {
  "code": 200,
  "data": {
    "filter_id": "7f3311e9f7ba4a16d3263e6e80f5eaa3"
  }
}

    HTTP Request

    POST https://coinwall.sectoken.io/api/v1/wall/filters

    请求参数

    参数名 参数类型 必填 说明
    name string 是 规则集名称
    coin string 是 币种信息
    address string 是 规则地址
    conditions array 是 规则条件

    conditions请求参数

    参数名 参数类型 必填 说明
    name string 是 条件名，如（transfer_vale, black_list）
    values array 是 条件内容, 填写 地址列表

    现有 conditions

    name values 示例 说明
    black_list ["2N7LGJUpCqufDE4NLjecHSimHCDrFwQsgrk", "2N2587eHcUCXpfjJ2B5bHQYNuijRtCXKpmi"] 黑名单列表
    white_list ["2N7LGJUpCqufDE4NLjecHSimHCDrFwQsgrk", "2N2587eHcUCXpfjJ2B5bHQYNuijRtCXKpmi"] 白名单列表
    freeze 1585312528 冻结时间
    transfer_value 1000000 单次转账金额
    transfer_day_total_value 1000000 单日转账金额
    transfer_day_limit 1000000 单日转账次数

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data array 返回内容

    data 参数

    参数名 参数类型 说明
    filter_id string 规则id

    HTTP Request

    GET https://coinwall.sectoken.io/api/v1/wall/filter

    URL参数

    参数名 参数类型 必填 说明
    coin string 是 币种信息

    请求头参数

    参数名 参数类型 必填 说明
    filter_id string 否 当币种为btc时需要填写, 其他币种暂时不需要
    change_address string 否 找零地址

    请求参数

    参数名 参数类型 必填 说明
    jsonrpc string 是 规则id
    id string 是 规则名称
    method string 是 固定填写 sendrawtransaction
    params string 是 签名完成的交易单信息

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data array 返回内容

    data 参数

    参数名 参数类型 说明
    txid string 交易单id
    is_sent bool 广播成功标记, (true: 成功, false: 失败)
