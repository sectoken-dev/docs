    Get the cost address of current account

    HTTP Request

    GET https://example.com/wallet/eth/cost/

    Request Parameters

    Parameter Name Type Required Description

    Response

    Parameter Name Type Description
    code int response code.If executed successfully code is 200, others are failures. All error codes
    data object response data.

    response data

    Parameter Name Type Description
    address string The cost address.
    balance string The balance of cost address(Unit is Wei).

    Create ETH Multi Wallet

    import requests

requests.post(
    utl="https://example.com/wallet/eth/getkey/", 
    headers={"HMAC": "your token"}, 
    json={
        "pubkeys": ["compressed pubkey1",...],
        "wallet_name": "your wallet name",
        "multi_type": "2-3",
        "order_type": 0,
    }
)


    
curl "https://example.com/wallet/eth/getkey/"
  -H "HMAC:your token"
  -H "Content-Type:application/json"
  -X POST
  -d '{"pubkeys": ["compressed pubkey1",...], \
      "wallet_name": "your wallet name", \
      "multi_type": "2-3", \
      "order_type": 0}'

    HTTP Request

    GET https://example.com/wallet/eth/getkey/

    Request Parameters

    Parameter Name Type Required Description
    pubkeys array true The array of public key.
    privkeys array false The array of private key.
    wallet_name string true The wallet name.
    password string false The wallet password.
    multi_type string true Multisignature type. Format {threshold}-{all}
    order_type int true Whether the platform participates in the signature. 0：participation，1：nonparticipation

    Response

    Parameter Name Type Description
    code int response code.If executed successfully code is 200, others are failures. All error codes
    data object response data.

    response data

    Parameter Name Type Description
    wallet_name string The wallet name.
    wallet_id string The wallet id.
    create_time string The wallet creation time.
    user_key_codes array The array of private key.
    pubkey_codes array The array of public key.
    activate_code string The wallet activate code.
    server_pub_key string The wallet x public key.

    Error Code

    Error code for failed execution.

    Code Description
    4203 Insufficient cost address balance.
    4401 Invalid Token.
    4403 The number of wallets reached the limit.
    4602 There is no cost address for the current account.
    500 Unknown Error.
