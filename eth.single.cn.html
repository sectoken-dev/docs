    创建eth单签钱包

    import requests

requests.post(
    url="https://testnet.sectoken.io/wallet/eth_single/create/", 
    headers={"HMAC": "your token"}, 
    json={
        "pubkey": "03976717038a5c6967535cc469acb6d7a43dee00568b9c4d4f2bbccb287503787c",
        "wallet_name": "your wallet name",
    }
)

    curl "https://testnet.sectoken.io/wallet/eth_single/create/" \
    -H "HMAC:your token" \
    -H "Content-Type:application/json" \
    -X POST \
    -d '{"pubkey": "03976717038a5c6967535cc469acb6d7a43dee00568b9c4d4f2bbccb287503787c", "wallet_name": "your wallet name"}'

    返回示例

    {
    "code": 200,
    "data": {
        "wallet_id": "cf761ad1b6c5f30a1557d942cb557b07",
        "create_time": "2019-10-10T10:13:03.195",
        "activate_code": "712968"
    }
}

    HTTP Request

    POST https://testnet.sectoken.io/wallet/eth_single/create/

    请求参数

    参数名 参数类型 必填 说明
    pubkey string 是 压缩公钥，长度为66
    privkey string 否 私钥，用户应自行加密传输，使用时解密。（必须使用base64编码）
    wallet_name string 是 钱包名

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data object 返回内容

    data 参数

    参数名 参数类型 说明
    wallet_id string 钱包的id
    create_time string 钱包创建时间
    activate_code string 钱包激活码

    激活钱包

    import requests

requests.post(
    url="https://testnet.sectoken.io/wallet/eth_single/activate/",
    headers={"HMAC": "your token"},
    json={
        "wallet_id": "your wallet id",
        "activate_code": "your activate code",
    }
)

    curl "https://testnet.sectoken.io/wallet/eth_single/activate/" \
    -H "HMAC:your token" \
    -H "Content-Type:application/json" \
    -X POST \
    -d '{"wallet_id": "your wallet id", "activate_code": "your activate code"}'

    返回示例

    {
    "code": 200,
    "data": {
        "address": "0x4b81124e2C5B6a774c0d4CFa0c346D37c5d9087B"
    }
}

    POST https://testnet.sectoken.io/wallet/eth_single/activate/

    activate_code string 是 钱包激活码

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data object 返回内容

    HTTP Request

    GET https://testnet.sectoken.io/wallet/eth_single/detail/

    请求参数

    参数名 参数类型 必填 说明
    wallet_id string 是 钱包id
    coin_type string 否 币种，默认为eth，可以指定为平台支持的erc20 token，如erc20_ht，erc20_koz

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data object 返回内容

    data 参数

    参数名 参数类型 说明
    wallet_id string 钱包id
    wallet_name string 钱包名
    address string 钱包地址
    balance string 钱包余额，单位为最小单位
    send_max string 单次转账上限，"0"为无上限
    send_day_limit integer 单日转账次数上限，0为无上限
    send_day_max string 单日转账上限，"0"为无上限
    freeze_overtime integer 钱包冻结截止时间，0为未冻结

    HTTP Request

    POST https://testnet.sectoken.io/wallet/eth_single/dotx/

    请求参数

    参数名 参数类型 必填 说明
    wallet_id string 是 钱包id
    to_address string 是 收款地址
    value string 是 转账金额，最小单位
    coin_type string 否 币种，默认为eth，可以指定为平台支持的erc20 token，如erc20_ht，erc20_koz
    feerate integer 否 转账速度，1快速，0普通，默认为0
    total integer 否 全部转出，置1生效，此时value值不生效。

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data object 返回内容

    data 参数

    参数名 参数类型 说明
    unsigned_tx string 未签名的交易
    privkey sting 创建钱包时提交的privkey字段，未提交则返回空字符串
