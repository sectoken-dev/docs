    Create Wallet

    import requests

requests.post(
    url="https://testnet.sectoken.io/wallet/eth_single/create/", 
    headers={"HMAC": "your token"}, 
    json={

        "wallet_name": "your wallet name",
    }
)

    curl "https://testnet.sectoken.io/wallet/eth_single/create/" \
    -H "HMAC:your token" \
    -H "Content-Type:application/json" \

    -d '{"pubkey": "03976717038a5c6967535cc469acb6d7a43dee00568b9c4d4f2bbccb287503787c", "wallet_name": "your wallet name"}'

    HTTP Request

    POST https://testnet.sectoken.io/wallet/eth_single/create/

    Request Parameters

    Parameter Name Type Required Description
    pubkey string Yes Compressed public key. Length of 66.
    privkey string Optional Private key. Users should encrypt and decrypt when using. (Must be base64 format.)
    wallet_name string Yes Wallet Name

    Response Parameters

    Parameter Name Type Description
    code integer Status code. Successful return 200. Others are Error Codes
    data object Response data.

    Data Parameters

    Parameter Name Type Description
    wallet_id string Wallet id
    create_time string Creation time of wallet
    activate_code string Activate code

    HTTP Request

    GET https://testnet.sectoken.io/wallet/eth_single/detail/

    Request Parameters

    Parameter Name Type Required Description
    wallet_id string Yes Wallet id
    coin_type string Optional Coin type. ETH or other ERC20 token supported by the platform. Default is "ETH".

    Response Parameters

    Parameter Name Type Description
    code integer Status code. Successful return 200. Others are Error Codes
    data object Response data.

    Data Parameters

    Parameter Name Type Description
    wallet_id string Wallet id
    wallet_name string Wallet Name
    address string Eth address of wallet
    balance string Balance. (Minimum unit)
    send_max string Maximum value per transfer. "0" is unlimited.
    send_day_limit integer Maximum number of transfers per day. 0 is unlimited.
    send_day_max string Maximum number of transfers per day. "0" is unlimited.
    freeze_overtime integer Wallet freeze deadline. 0 is not frozen.

    Request Parameters

    Parameter Name Type Required Description
    wallet_id string Yes Wallet id
    to_address string Yes Destination address.

    privkey string The privkey field that was submitted when the wallet was created. Uncommentted returns an empty string.

    Add Wallet WebHook

    Add a webhook that will result in an HTTP callback at the specified URL from Sectoken when events are triggered.

    There is a limit of 10 webhooks per wallet.

    import requests

    Transaction Callback

    When a transaction occurs in a wallet，Sectoken will send an Http POST request to the specified url，The content-type of the request header is application/json. Need to add the wallet Webhook first.

    This url is specified by the user, who must deploy the service to receive the HTTP POST request.

    Sectoken servers will make a POST http request to the URL defined with a JSON payload, and expect a HTTP 200 OK. If a successful response is not received, Sectoken will attempt to retry the webhook with an increasing delay between each retry.
