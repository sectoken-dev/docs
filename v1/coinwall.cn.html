    创建规则集

    curl "https://wall.sectoken.io/api/v1/wall/add_filter/" \
    -H "Authorization: Bearer $BEARER_TOKEN" \  
    -X POST \
    -d '{
           "name": "test4two",
           "coin": "eth",
           "conditions": [
                           {
                              "name": "transfer_value",
                              "values": ["10000"]
                           },
                           {
                             "name": "black_list",
                             "values": [
                                          "0x3cd751e6b0078be393132286c442345e5dc49699",
                                          "0x25f3e46cfdbcc9df989c98254eac673db5a50084"
                                        ]
                            }
                         ],
            "token_contract": "0xdAC17F958D2ee523a2206206994597C13D831ec7"
        }'

    返回示例

    {
  "code": 200,
  "data": {
    "filter_id": "7f3311e9f7ba4a16d3263e6e80f5eaa3"
  }
}

    HTTP Request

    POST https://wall.sectoken.io/api/v1/wall/add_filter/

    请求参数

    参数名 参数类型 必填 说明
    name string 是 规则集名称
    coin string 是 币种信息
    conditions array 是 规则条件
    token_contract string 否 为空表示主链币eth/bsc

    conditions请求参数

    参数名 参数类型 必填 说明
    name string 是 条件名,如transfer_vale, black_list
    values array 是 条件内容:见下表

    现有 conditions

    name values 示例 说明
    black_list ["0x25f3E46CfdBcC9DF989c98254eaC673Db5A50084", "0x25f3E46CfdBcC9DF989c98254eaC673Db5A50084"] 黑名单列表
    white_list ["0x973f18d029deC635873543535C45e8E2fB7B37Ab", "0x057885D1F891741B5a8Ab4060880056eF70C4807"] 白名单列表
    transfer_value ["1000000"] 单次转账金额,币种最小粒度位数表示
    transfer_day_total_value ["5000000"] 单日转账金额,币种最小粒度位数表示
    transfer_day_limit ["5"] 单日转账次数

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data dict 返回内容

    data 参数

    参数名 参数类型 说明
    filter_id string 规则集id

    HTTP Request

    GET https://wall.sectoken.io/api/v1/wall/filters/

    请求参数

    参数名 参数类型 必填 说明
    filter_id string 否 规则集id
    coin string 否 币种名: eth,bsc
    page int 否 默认1
    page_size int 否 默认10

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data array 返回内容

    data 参数

    参数名 参数类型 说明
    name string 规则集名称
    filter_id string 规则集Id
    coin string 币种名
    token_contract string erc20/bep20 合约地址，为空表示主链币eth/bsc
    status int 规则状态 : 1启用
    coindtions array 规则条件

    HTTP Request

    GET https://wall.sectoken.io/api/v1/wall/history/

    请求参数

    参数名 参数类型 必填 说明
    filter_id string 是 规则id
    page int 否 默认1
    page_size int 否 默认10

    返回参数

    参数名 参数类型 说明
    code integer 返回码，200成功，其他失败，请查看错误码
    data array 返回内容

    data 参数

    参数名 参数类型 说明
    coin string 币种
    from array 来源地址
    to array 目的地址
    value string 转账金额
    decimal int 币种的小数位
    blocked int 是否被禁止广播，(0: 正常, true:被禁止)
    condition_name string 如果被禁止广播，禁止的条件名
    condition_value string 如果被禁止广播，禁止的条件内容
    timestatmp int 时间戳

    HTTP Request

    GET https://wall.sectoken.io/api/v1/wall/broadcast/

    URL参数

    参数名 参数类型 必填 说明
    coin string 是 主链币种名

    请求头参数

    参数名 参数类型 必填 说明
    filter_id string 否 规则集ID

    请求参数

    参数名 参数类型 必填 说明
    jsonrpc string 否 2.0
    id string 是 客户端任意字符串
    method string 是 eth/bsc填eth_sendRawTransaction
    params string 是 签名完成的交易单信息

    不检查广播交易单

    curl "https://wall.sectoken.io/api/v1/wall/unchecked_broadcast/$coin" \
    -H "Authorization: Bearer $BEARER_TOKEN" \  
    -x POST \
    -d '{
        "jsonrpc": "2.0",
        "id": "curltest",
        "method": "sendrawtransaction",
        "params": [
        "$signed_tx"
            ]
        }'

    返回示例

    {
    "jsonrpc": "2.0", 
    "id": "curltest", 
    "result": "0x9a5dc1bb2e7dc22ca3eddfe6d62f4f865a4bff12990b34b0a650bcb233f1bb20"
}

    HTTP Request

    POST https://wall.sectoken.io/api/v1/wall/unchecked_broadcast/$coin

    URL参数

    参数名 参数类型 必填 说明
    coin string 是 币种信息

    请求参数

    参数名 参数类型 必填 说明
    jsonrpc string 是 2.0
    id string 是 客户端自定义字符串
    method string 是 eth/bsc填eth_sendRawTransaction
    params string 是 签名完成的交易单信息

    错误码

    当发生错误时会返回错误码

    错误码 错误说明
    4000 请求参数错误
    5000 未知错误
    4203 超出规则集个数
